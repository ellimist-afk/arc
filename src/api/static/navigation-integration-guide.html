🔧 Integration Instructions

1. Replace Existing CSS

<!-- Remove these conflicting files --> <link rel="stylesheet" href="css/navigation-enhanced.css"> <link rel="stylesheet" href="css/navigation-effects.css"> <link rel="stylesheet" href="css/navigation-fix.css"> <!-- Replace with modern navigation --> <link rel="stylesheet" href="css/navigation-modern.css">

2. Update HTML Structure

<!-- OLD: Multiple class system --> <header class="tbx-header theme-v2"> <a class="nav-link active">Dashboard</a> </header> <!-- NEW: BEM methodology --> <nav class="tbx-nav" role="navigation"> <a class="tbx-nav__item tbx-nav__item--active">Dashboard</a> </nav>

3. JavaScript Integration

// Add active state management const navItems = document.querySelectorAll('.tbx-nav__item'); navItems.forEach(item => { item.addEventListener('click', (e) => { // Remove active from all items navItems.forEach(i => i.classList.remove('tbx-nav__item--active')); // Add active to clicked item e.target.classList.add('tbx-nav__item--active'); }); }); // Mobile menu toggle const mobileToggle = document.querySelector('.tbx-nav__mobile-toggle'); const mobileMenu = document.querySelector('.tbx-nav__mobile-menu'); mobileToggle.addEventListener('click', () => { const isOpen = mobileMenu.classList.contains('tbx-nav__mobile-menu--open'); mobileMenu.classList.toggle('tbx-nav__mobile-menu--open'); mobileToggle.setAttribute('aria-expanded', !isOpen); });

🎨 Design Tokens (CSS Custom Properties)

All styling is controlled through CSS custom properties in :root. This enables easy theming and consistent design across the application.
:root { /* Navigation Colors */ --nav-bg-primary: #0E0E11; --nav-accent: #9146FF; --nav-text-primary: #FFFFFF; /* Spacing System */ --nav-space-sm: 0.5rem; --nav-space-md: 0.75rem; --nav-space-lg: 1rem; /* Typography */ --nav-font-size-base: 0.9375rem; --nav-font-weight-medium: 500; /* Animation */ --nav-transition-fast: 150ms cubic-bezier(0.4, 0, 0.2, 1); }
